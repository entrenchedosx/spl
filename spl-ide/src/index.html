Welcome to SPL

Simple Programming Language – a small, friendly language to learn coding.

This IDE runs SPL scripts only. Use the toolbar or F5 to run your code.

Quick start
  • Press F5 or click Run to run your code
  • Ctrl+Space → IntelliSense: type pri for print(), or math. for sqrt, sin, PI, etc.
  • Syntax errors show while typing (run spl --check under the hood)
  • Matching braces { } are highlighted; Ctrl+Shift+O for outline
  • Use Snippets in the sidebar to insert loops, functions, and more
  • Open Docs (toolbar) for built-ins and import math, import string, etc.
  • Ctrl+Shift+P → command palette (Zoom, Fold, Theme, Shortcuts…)
